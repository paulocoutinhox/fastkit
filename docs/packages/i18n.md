# fastkit-i18n

The single translation authority.

Full detail: [Errors and i18n](../concepts/errors-and-i18n.md).

## Highlights

- **`Translator`** merges catalogs and resolves a locale's full catalog via `messages(locale)`.
- **`catalogs.BASE_CATALOGS`** — every framework `error.*` + `validation.*` string, en + pt complete.
- Keys are **`context.local`, kebab-case local part** (`error.invalid-email`, `grid.apply`).
- `LocaleResolver` consults `translator.supported()` live and falls back region → base
  (`pt-BR` → `pt`). `Accept-Language` with a bad `q=` value is skipped, never a 500.

## Extend

```python
def register_translations(self, context):
    context.component("translator").add_catalog("es", {"login.submit": "Entrar"})
```

A registered locale is immediately resolvable from `Accept-Language`/cookie. See
[Add a locale / translations](../guides/add-locale.md).
